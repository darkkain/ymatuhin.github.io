В первой статье хотелось бы традиционно написать "Hello World", можно было бы сделать и вот так: console.log('hello world');, но это, как мне кажется слишком простой путь. Поэтому я поискал интересные Hello World на javascript, и то что я нашел меня по началу немного обескуражело, а потом вызвало невероятное желание разобраться в чем-же все-таки дело.

Не буду долго томить, вот пример того, что я нашел:

for(i=s="";l=[43117609029,64070269789,46349920852,46890400349][i++];)for(j=0,s+="\n";c=l.toString(2)[j++];)s+=" #"[c]

А вот результат который можно увидеть в консоли:

# #     # #       # #        #   # #
### ### # # ###   # # ### ## # ### #
# # ##  # # # # # ### # # #  # # #
# # ### # # ###   # # ### #  # ### #

Так как же все-таки работает этот кусок кода? Сейчас узнаем :)

Для начала отформатируем этот кусочек

for (i = s = ""; l = [43117609029, 64070269789, 46349920852, 46890400349][i++];)
	for (j = 0, s += "\n"; c = l.toString(2)[j++];) s += " #"[c]

Тут мы видим два вложенных цикла, первый цикл - необычный, внутри него l на каждой итерации равняется одному из чисел в массиве (лично я первый раз увидел такой пример).

Во вложенном цикле мы видим похожую конструкцию, только в ней, переменная c будет равна символу из двоичного представления числа l. При каждой новой инициализации этого цикла мы создаем новую строку s += "\n", а на каждой итерации мы ставим один из символов пробел или # s += " #"[c]

Вот так, благодаря обычным двоичным числам у нас получился интересный ASCII art.
