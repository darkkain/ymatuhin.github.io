Фронтенд дайджест #1 — Awesome lists, bower, regexp, RSS, Git

11 августа 2015, обновлено

Когда я нахожу интересные ссыки, то хочу ими поделиться. Поэтому я собираю эти ссылки в Wunderlist, а когда набирается 15, публикую в дайджесте. Поехали…

  1. Испытывали ли вы чувство, когда непонятно как назвать переменную для цвета? Если да, то сервис Name that Color вам пригодится. Вставляете hex код цвета, и получаете название для него.

  2. Рейтинг CSS библиотек по популярности.

  3. GUI для кривых безье.

  4. typebase.css — таблица стилей для типографики. Пробовал на сайте, мне не понравилось. Стартовая точка для типографики в веб.

  5. Get awesomeness — список всего по всему.

  6. Главред помогает писать текст в информационном стиле. Пользуюсь перед каждой публикацией. И подписался на рассылку.

  7. Приходилось ли вам делать поле для ввода, которое тянется вместе с текстом? Stretchy сделает это за вас.

  8. Дополнительный переносной экран для ноутбука. Сама идея хороша, но дома я и обычным не пользуюсь. Только на работе.

  9. Хотел написать статью о Bower, но наткнулся на пост Артема — «Bower: зачем фронтенду нужен менеджер пакетов», к которой нечего добавить.

  10. Пишите регулярки в консоли браузера? Как на счет попробовать онлайн сервис regexr? В нем это делать куда удобнее и есть шпаргалка по регуляркам, которую я каждый раз гуглю.

  11. Делаете микроразметку для сайта? Schema Creator поможет вам. Есть генераторы и примеры для многих типов. Жаль для блога нет.

  12. Читаете о фронтенде по RSS и незнаете что еще почитать? Тогда этот список RSS-фидов для вас.

  13. Избегаете гит через консоль? Пользуетесь только GUI? Не бояться консоли поможет legit, который делает понятные алиасы над гитом.

  14. У вас есть сайт? Или вы продвигаете свой проекта? SEO чеклист поможет вам ничего не забыть.

  15. Онлайн конвертер изображений в WebP.

У тебя есть пара крутых ссылок? Поделись ими в комментариях.
