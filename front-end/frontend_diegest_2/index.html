Опубликовано: 21 августа, 2015, обновлено сегодня.

Когда я нахожу интересные ссыки, то хочу ими поделиться. Поэтому я собираю эти ссылки в Wunderlist, а когда набирается 15, публикую в дайджесте. Поехали...

  1. БЭМ с каждым годом набирате популярность. Однако многие пишут плохой БЭМ. Игорь Зенич в презентации «Пишем БЭМ правильно» показывает как надо.

  2. Пользуетесь Pocket? Уверен что у вас изрядная стопка полезных статей там. Александр Бизиков отсортировал свой Pocket и делится получившимся списком из 81 статьи с нами.

  3. Онлайн конструктор для CSS матриц.

  4. Коллекция из 500 SVG логотипов популярных компаний и сервисов.

  5. Полифил для Emoji компании Twitter — Twemoji.

  6. Новая сборка Vivaldi 1.0.249.12. Добавили веб панели, кнопки для расширений, прогресс загрузки в адресной строке и другое.

  7. Paste App — клипбоард менеджер для Mac OS с поиском по истории. Сохраняет в истории не только текст, но и файлы.

  8. Вчера вышел Bootstrap v4.0.0-alpha переписанный на Sass. Мне сложно запомнить элементы из этого фреймворка, поэтому я нашел страницу с элементами. Открываем инспектор, копируем код к себе, наслаждаемся.

  9. Библиотека для 2D векторов — VictorJS.

  10. Кроссплатформенные приложения на web стеке? Github Electron поможет. На нем написан Atom, Slack, Avocode.

  11. Рассуждения Максима Ильяхова о вопросительном знаке в заголовке.

  12. Словарь терминов по фронтенду редакции «Веб-стандартов».

  13. HMTL 5 Please — покажет какие фичи из HTML5 и CSS3 можно использовать и что для этого требуется.

  14. Учебник по холсту в HTML5 (canvas).

  15. Шпаргалка для свойств и методов HTML5 Холста.


У тебя есть пара крутых ссылок? Поделись ими в комментариях.
